# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
|   0.2   | :white_check_mark: | (Unreleased)
|   0.1   | :x:                |

## Reporting a Vulnerability

To report a vulnerability, contact me privately via discord (unipm) and tell me
1. what the vulnerability is
2. how to replicate it
3. what version of LCEAuth was used

After reporting a vulnerability, I will spend any spare time I have towards fixing the vulnerability.
